# Style Guide: wpf-xaml

## Scope
WPF XAML markup for application and windows.

## Observed Conventions
- App uses `StartupUri` for main window.
- Window markup is minimal with a root `Grid`.
- No resources or styles defined yet.

## Examples
- `./src/EZSampler.UI.Wpf/App.xaml`
- `./src/EZSampler.UI.Wpf/MainWindow.xaml`
