# Style Guide: wpf-codebehind

## Scope
Code-behind for WPF app and windows.

## Observed Conventions
- Minimal partial classes that only call `InitializeComponent()`.
- No custom startup logic or event handlers yet.

## Examples
- `./src/EZSampler.UI.Wpf/App.xaml.cs`
- `./src/EZSampler.UI.Wpf/MainWindow.xaml.cs`
